# Preferencias Del Proyecto (Codex)

Estas preferencias son para las sesiones de desarrollo en este repositorio.

## Preferencia principal del usuario

- Usar subagentes por defecto (explorer/worker) siempre que ayude a avanzar mas rapido.
- Priorizar trabajo en paralelo cuando haya tareas independientes.
- Al iniciar una sesion, leer este archivo para tomar contexto operativo.

## Nota

- Si una tarea es chica o bloqueante en el camino critico, resolverla directamente en el agente principal para no agregar latencia innecesaria.
